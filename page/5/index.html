Windows平台

VSCode

优点：

  • 界面美观
  • 扩展强大
  • 代码高亮
  • 启动速度快

缺点：

  • AHK中文无法对齐
  • Alt多选需要在设置中设置为Ctrl/Command，体验极差

Notepad++

缺点：

  • 界面不美观
  • 文件不能自动关联

Sublime Text 3

缺点：

  • Windows不支持Alt多选
  • 闭源

AutoHotkey在热键体验不好的地方总结

发表于 2019-04-14 分类于 效率提升
本文字数： 1.5k 阅读时长 ≈ 1 分钟

最开始的想法

  • 长按字母键打开常用应用，不影响字母输入
  • 长按鼠标中键打开指定应用，不影响鼠标中键原有功能
  • 双击Alt唤醒Wox，不影响Alt原有功能

失败后的备选方案

  • 使用RShift + 字母打开常用应用
  • 使用Wgesture，定义按下鼠标右键+单击鼠标左键打开指定应用（已实现）
  • 使用Capslock + F发送Alt+Space，实现唤醒Wox（已实现）
