2018-9-2：介绍我当前使用的Chrome插件

插件列表

  • Lastpass：在线密码管理器

  • Tampermonkey：js脚本管理器

  • 广告净化器：广告过滤，特色功能是过滤视频广告。

  • 云盘万能钥匙：自动获取并填写百度网盘密码（类似WiFi万能钥匙）

  • Adblock：用于去除广告净化器不能很好过滤的广告。

  • Gitzip for github：双击文件或者文件夹下载Github文件

  • Extension Manager (aka switcher)：快速关闭无用插件，无需进入扩展管理关闭。

  • Fireshot：网页截图，支持长截图，只需截图时滚动鼠标即可。
